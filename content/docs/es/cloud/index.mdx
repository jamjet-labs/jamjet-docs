# JamJet Cloud

JamJet Cloud es la capa de gobernanza alojada para agentes de IA. Integra dos líneas de SDK y obtienes aplicación de políticas, seguimiento de presupuesto, flujos de aprobación y un registro de auditoría — para cualquier aplicación de IA, en cualquier framework.
