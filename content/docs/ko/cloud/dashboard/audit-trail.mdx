# 감사 추적

감사 추적은 JamJet Cloud가 프로젝트에 대해 수신한 모든 스팬의 완전하고 불변한 기록입니다. 모든 LLM 호출, 모든 도구 실행, 모든 정책 결정, 모든 승인 결과가 여기에 기록됩니다. 검색, 필터링, 내보내기가 가능하여 규정 준수 검토, 장애 사후 분석, 비용 조사를 위한 주요 인터페이스입니다.

## 스팬 검색

감사 추적 상단의 검색창은 에이전트 이름, 모델 식별자, 사용자 ID, 정책 이름, 스팬 메타데이터와 일치하는 자유 텍스트 쿼리를 받습니다. `gpt-4o`를 입력하면 해당 모델을 사용한 모든 스팬이 반환됩니다. 에이전트 이름을 입력하면 해당 에이전트가 발생시킨 모든 스팬이 반환됩니다. [사용자 컨텍스트](/cloud/telemetry/user-context)를 사용하는 경우 사용자 ID를 입력하면 해당 사용자에게 귀속된 모든 항목이 반환됩니다.

검색 결과는 기본적으로 최신순으로 정렬됩니다. 열 헤더를 사용하여 순서를 반대로 하거나 비용 또는 지연 시간별로 정렬할 수 있습니다.

## 필터

필터는 텍스트 검색 전후에 결과 집합을 좁힙니다. 필터는 조합 가능하며, 활성화된 모든 필터가 동시에 적용됩니다.

**스팬 속성:**
- 에이전트 — 하나 이상의 명명된 에이전트.
- 사용자 — 특정 `user_id` (SDK에서 사용자 컨텍스트 필요).
- 모델 — 예: `gpt-4o`, `claude-3-7-sonnet`, `gemini-2.5-pro`.
- 비용 — 스팬당 USD 단위의 최소 및/또는 최대 비용.
- 시간 범위 — 절대 타임스탬프 또는 상대적 시간 윈도우 (최근 1시간, 24시간, 7일 또는 사용자 지정).

**승인 상태:**
- `approved` — `requireApproval`을 통해 사람이 작업을 승인함.
- `rejected` — 사람이 작업을 거부함.
- `timed_out` — 사람이 조치하기 전에 승인 요청이 만료됨.
- `not_required` — 승인 게이트를 트리거하지 않고 스팬이 완료됨.

**정책 결정:**
- `allow` — 정책이 평가되어 도구 호출을 허용함.
- `block` — 정책이 실행 전에 도구 호출을 차단함.
- `require_approval` — 정책이 사람 게이트로 에스컬레이션됨.

`block` 정책 필터를 특정 에이전트 필터와 결합하면 "이 에이전트가 시도했지만 정책이 차단한 도구 호출은 무엇인가?"라는 질문에 가장 빠르게 답할 수 있습니다.

## 스팬 상세 정보

감사 추적의 행을 클릭하면 스팬 상세 패널이 열립니다. 다음 정보를 표시합니다:

- 모델로 전송된 전체 요청 페이로드(데이터 마스킹을 구성한 경우 마스킹됨).
- 도구 호출 및 인수를 포함한 전체 응답.
- 타이밍 분석: 첫 토큰까지의 시간, 전체 지연 시간, 프롬프트 및 완료 토큰 수.
- 호출 시점의 모델 공개 토큰 가격으로 계산된 USD 비용.
- 해당 스팬에 대해 평가된 정책 결정, 일치하는 규칙 및 결과.
- 승인 기록(필요한 경우) — 누가 승인 또는 거부했는지, 언제 승인/거부했는지, 입력한 거부 사유.
- 에이전트, 사용자, 환경 및 프로세스 컨텍스트 속성.

## 내보내기

결과 테이블 상단의 **내보내기** 버튼을 사용하면 CSV 또는 JSON 형식으로 다운로드할 수 있습니다. 내보내기는 활성화된 모든 필터와 현재 검색 쿼리를 반영합니다 — 화면에 표시된 내용이 내보내지는 내용입니다.

내보내기 사용 사례:

- **SOC 2 감사 기간.** 시간 범위를 감사 기간으로 설정하고, 선택적으로 프로덕션으로 필터링한 후 JSON으로 내보냅니다. 결과는 해당 기간의 모든 모델 상호작용을 비용, 모델, 에이전트 및 정책 결과를 포함하여 기계 판독 가능한 형태로 기록한 것입니다.
- **인시던트 사후 분석.** 인시던트가 발생한 좁은 시간 범위를 내보내 오프라인 분석 또는 사후 분석 문서 첨부에 사용합니다.
- **청구 조정.** 청구 기간의 모든 스팬을 내보내고 `cost_usd`를 합산하여 모델 제공업체의 청구서와 대조 확인합니다.

결과 세트가 약 10,000개 스팬 미만인 경우 내보내기는 동기식으로 처리됩니다. 더 큰 내보내기는 대기열에 추가되며 계정 이메일로 전송된 다운로드 링크를 통해 제공됩니다.

## 보관

스팬 보관 기간은 플랜 등급에 따라 다릅니다:

- **무료 등급** — 7일 롤링 보관. 7일이 지난 스팬은 자동으로 삭제됩니다.
- **유료 등급** — 더 긴 보관 기간을 제공하며, **설정 → 데이터 및 보관**에서 구성할 수 있습니다. 정확한 보관 한도는 [app.jamjet.dev](https://app.jamjet.dev)의 현재 플랜 비교를 확인하세요. 보관 한도는 변경될 수 있습니다.

> 보관 기간을 초과하여 스팬을 보존해야 하는 경우, 기한 전에 내보내기를 진행하세요. JamJet은 삭제 후 소급 내보내기를 지원하지 않습니다.
