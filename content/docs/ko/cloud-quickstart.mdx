# JamJet Cloud — 개발자 빠른 시작

두 줄의 코드로 모든 OpenAI 또는 Anthropic 에이전트에 거버넌스, 비용 추적, 감사 로그를 추가하세요.

**5분 이내**에 추적 정보를 보고하는 계측된 에이전트를 구축할 수 있습니다.

> **참고:** JamJet Cloud는 오픈소스 JamJet 런타임 + Engram과 함께 제공되는 호스팅 유료 제품입니다. 오픈소스 JamJet은 영구 무료이며, Cloud는 멀티 테넌트 대시보드, 보관된 감사 로그, 호스팅 메모리, 정책/승인 인터페이스를 추가합니다. 출시된 기능과 향후 계획은 [로드맵](https://jamjet.dev/cloud)을 참조하세요.

---

## 1. API 키 발급

1. [**app.jamjet.dev**](https://app.jamjet.dev)에서 로그인하세요 (GitHub OAuth, Google OAuth 또는 매직 링크).
2. **Settings → Projects**를 열고 프로젝트를 생성하세요.
3. 생성 시 한 번만 표시되는 API 키를 복사하세요. `jj_xxxxxxxxxxxx` 형식입니다.

> 키는 나중에 다시 조회할 수 없습니다. 분실한 경우 새 프로젝트를 생성하세요.

---

## 2. SDK 설치

```bash
pip install jamjet
```

`jamjet.cloud` 서브모듈은 **0.6.0**부터 메인 `jamjet` 패키지에 포함되어 있습니다.

선택사항: 사용하는 LLM SDK를 설치하면 JamJet이 자동으로 계측합니다.

```bash
pip install jamjet[openai]      # OpenAI 자동 계측
pip install jamjet[anthropic]   # 또는 Anthropic
pip install jamjet[cloud-all]   # 둘 다
```

---

## 3. 초기화 및 실행

프로세스 시작 시 LLM 호출 전에 두 줄을 추가하세요:

```python
import jamjet.cloud as jamjet

jamjet.configure(api_key="jj_xxxxxxxxxxxx", project="my-agent")
```

이제 기존 OpenAI 호출이 자동으로 캡처됩니다:

```python
from openai import OpenAI

client = OpenAI()
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "hello"}],
)
```

[**app.jamjet.dev/dashboard/traces**](https://app.jamjet.dev/dashboard/traces)를 열면 약 5초 이내에 모델, 토큰 수, 비용, 실행 시간과 함께 호출 정보가 표시됩니다.

## 4. 거버넌스 추가 (선택 사항)

### 이름으로 도구 차단

```python
jamjet.policy("block", "payments.*")
jamjet.policy("require_approval", "delete_*")
```

차단된 도구는 모델이 보기 전에 필터링됩니다. `require_approval`은 대시보드에서 사람이 승인할 때까지 실행을 일시 중지합니다.

### 지출 한도 설정

```python
jamjet.budget(max_cost_usd=5.00)
```

예산을 초과하는 호출은 실행 대신 `BudgetExceeded`를 발생시킵니다.

### 자체 함수 추적

```python
@jamjet.trace
def lookup_customer(customer_id: str) -> dict:
    ...
```

각 호출은 주변 LLM 호출과 동일한 추적의 스팬이 됩니다.

### 사람 개입 승인

```python
approval_id = jamjet.require_approval(
    action="charge_card",
    context={"amount_usd": 200, "customer": "cust_42"},
    timeout_seconds=300,
)
```

대시보드의 **승인** 섹션에서 검토자가 승인하거나 거부할 때까지 차단됩니다.

## 5. 구성

`jamjet.configure(...)`는 다음을 허용합니다:

| 인자 | 기본값 | 비고 |
|---|---|---|
| `api_key` | 필수 | 프로젝트의 `jj_...` 키 |
| `project` | `"default"` | 추적의 논리적 그룹화 |
| `api_url` | `https://api.jamjet.dev` | 자체 호스팅 시 재정의 (사용 가능 시) |
| `auto_patch` | `True` | OpenAI/Anthropic 자동 계측을 건너뛰려면 비활성화 |
| `flush_interval` | `5.0` (초) | 백그라운드 스레드가 배치를 전송하는 주기 |
| `flush_size` | `50` | 즉시 전송을 트리거하는 배치 크기 |
| `capture_io` | `False` | `True`로 설정하면 전체 프롬프트/응답 페이로드 캡처 |

동일한 옵션은 환경 변수로 설정할 수 있습니다: `JJ_API_KEY`, `JJ_PROJECT`, `JJ_API_URL`.

---

## 6. 전송되는 데이터 (그리고 전송되지 않는 데이터)

기본적으로 JamJet Cloud는 **메타데이터**를 캡처합니다: 모델 이름, 토큰 수, 지연 시간, 비용 추정치, 도구 이름, 상태. 프롬프트 및 응답 *내용*은 `capture_io=True`로 설정하지 않는 한 전송되지 **않습니다**.

SDK는 **페일 오픈** 방식입니다: `api.jamjet.dev`에 접근할 수 없는 경우에도 에이전트는 계속 실행됩니다. 실패한 배치는 지수 백오프로 재시도되며 5회 연속 실패 후 삭제됩니다 (서킷 브레이커).

## 7. 다음 단계

활발히 개발 중입니다. 전체 일정은 [로드맵](https://jamjet.dev/cloud)을 참조하세요.

- **멀티 에이전트 가시성** — 에이전트 식별, 에이전트 간 트레이스 전파, 에이전트 통신 네트워크 그래프 뷰 (1단계 / 2026년 3분기).
- **Java SDK** — Spring AI / LangChain4j용 동일한 드롭인 방식 (1단계).
- **중앙 집중식 정책 집행** — 서버 측 정책 결정, 감사된 위임 체인.
- **호스팅 메모리** — Cloud에 번들로 제공되는 Engram, 에이전트별 범위 지정, 에이전트 플릿 전체에서 공유.
- **재생** — 대시보드에서 입력 기록과 함께 모든 트레이스 재실행.
- **OTel GenAI 수집** — SDK 마이그레이션 없이 기존 Phoenix / OpenLLMetry / Langfuse로 계측된 앱을 JamJet으로 연결.

## 문제 해결

| 증상 | 원인 |
|---|---|
| 호출이 표시되지 않음 | 5초 대기 (플러시 간격). 여전히 표시되지 않으면 API 키가 프로젝트와 일치하는지 확인. |
| `RuntimeError: JamJet Cloud not configured` | 첫 LLM 호출 전에 `jamjet.configure()`가 호출되지 않음. |
| `BudgetExceeded` | 호출이 `budget(max_cost_usd=...)`로 설정된 누적 비용 초과. 증액하거나 제거. |
| Anthropic 호출이 캡처되지 않음 | `pip install jamjet[anthropic]` — 자동 패칭은 SDK를 임포트할 수 있어야 작동. |
| OpenAI 호출이 캡처되지 않음 | `pip install jamjet[openai]`. SDK는 클래스 레벨에서 `OpenAI().chat.completions.create`를 패치 — 모듈 레벨과 인스턴스 사용 모두 캡처됨. |
| API에서 401 오류 | 키가 교체되었거나 잘못됨. [app.jamjet.dev/dashboard/settings](https://app.jamjet.dev/dashboard/settings)에서 새 프로젝트 생성. |
| API에서 429 오류 | 속도 제한 중. 기본값은 API 키당 초당 100건. 증량이 필요하면 Discord 또는 `support@jamjet.dev`로 문의. |
