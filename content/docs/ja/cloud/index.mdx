# JamJet Cloud

JamJet Cloudは、AIエージェントのためのホスト型ガバナンスレイヤーです。SDKを2行追加するだけで、ポリシー適用、予算追跡、承認フロー、監査証跡を、あらゆるフレームワーク上のあらゆるAIアプリに導入できます。
