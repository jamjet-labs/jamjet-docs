# Cloud Integrations

JamJet Cloud's TypeScript and Python SDKs work with any framework that speaks OpenAI / Anthropic protocols. For frameworks with their own abstractions, dedicated integrations give you cleaner enforcement and richer telemetry.

  <Card title="LangChain4j" href="/integrations/langchain4j" description="Java agents and tools. Advisor pattern." />
  <Card title="Spring Boot" href="/integrations/spring-boot" description="Auto-config starter. Engram ChatMemoryStore." />
  <Card title="Mastra / Express / CF Workers" href="/integrations" description="Coming soon." />
</Cards>

> Each integration is a thin adapter on top of `@jamjet/cloud`. If your framework isn't listed, the bare SDK works too.
