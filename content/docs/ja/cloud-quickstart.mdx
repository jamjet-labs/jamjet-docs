# JamJet Cloud — 開発者向けクイックスタート

OpenAIまたはAnthropicのエージェントに、ガバナンス、コスト追跡、監査証跡を2行で追加できます。

**5分以内**にトレースを報告する計装済みエージェントが動作します。

> **注意:** JamJet Cloudは、オープンソースのJamJetランタイム + Engramと併存する有料のホスティング製品です。オープンソース版JamJetは永久に無料です。Cloudはマルチテナントダッシュボード、保持された監査証跡、ホスト型メモリ、ポリシー/承認機能を追加します。出荷済みの機能と今後の予定については[ロードマップ](https://jamjet.dev/cloud)を参照してください。

---

## 1. APIキーを取得

1. [**app.jamjet.dev**](https://app.jamjet.dev)にサインイン(GitHub OAuth、Google OAuth、またはマジックリンク)
2. **Settings → Projects**を開き、プロジェクトを作成
3. 作成時に一度だけ表示されるAPIキーをコピー。形式は`jj_xxxxxxxxxxxx`

> キーは後で取得できません。紛失した場合は新しいプロジェクトを作成してください。

---

## 2. SDKをインストール

```bash
pip install jamjet
```

`jamjet.cloud`サブモジュールは**0.6.0**以降のメイン`jamjet`パッケージに同梱されています。

オプション:使用するLLM SDKをインストールすると、JamJetが自動計装します。

```bash
pip install jamjet[openai]      # OpenAIを自動計装
pip install jamjet[anthropic]   # またはAnthropic
pip install jamjet[cloud-all]   # 両方
```

---

## 3. 初期化と実行

LLM呼び出しの前に、プロセス開始時に2行追加:

```python
import jamjet.cloud as jamjet

jamjet.configure(api_key="jj_xxxxxxxxxxxx", project="my-agent")
```

既存のOpenAI呼び出しが自動的にキャプチャされます:

```python
from openai import OpenAI

client = OpenAI()
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "hello"}],
)
```

[**app.jamjet.dev/dashboard/traces**](https://app.jamjet.dev/dashboard/traces)を開くと、モデル、トークン数、コスト、実行時間とともに約5秒以内に呼び出しが表示されます。

---

## 4. ガバナンスを追加する（オプション）

### ツール名によるブロック

```python
jamjet.policy("block", "payments.*")
jamjet.policy("require_approval", "delete_*")
```

ブロックされたツールは、モデルが認識する前に除外されます。`require_approval`は、ダッシュボードで人間が承認するまで実行を一時停止します。

### 支出上限の設定

```python
jamjet.budget(max_cost_usd=5.00)
```

予算を超過する呼び出しは、実行される代わりに`BudgetExceeded`を発生させます。

### 独自の関数をトレースする

```python
@jamjet.trace
def lookup_customer(customer_id: str) -> dict:
    ...
```

各呼び出しは、周辺のLLM呼び出しと同じトレース内のスパンとして記録されます。

### ヒューマン・イン・ザ・ループ承認

```python
approval_id = jamjet.require_approval(
    action="charge_card",
    context={"amount_usd": 200, "customer": "cust_42"},
    timeout_seconds=300,
)
```

ダッシュボードの**承認**画面でレビュアーが承認または拒否するまでブロックします。

## 5. 設定

`jamjet.configure(...)`は以下を受け付けます：

| 引数 | デフォルト | 備考 |
|---|---|---|
| `api_key` | 必須 | プロジェクトの`jj_...`キー |
| `project` | `"default"` | トレースの論理的グループ分け |
| `api_url` | `https://api.jamjet.dev` | セルフホスト用のオーバーライド（利用可能時） |
| `auto_patch` | `True` | OpenAI/Anthropicの自動インスツルメンテーションをスキップする場合は無効化 |
| `flush_interval` | `5.0`（秒） | バックグラウンドスレッドがバッチを送信する頻度 |
| `flush_size` | `50` | 即座に送信をトリガーするバッチサイズ |
| `capture_io` | `False` | `True`の場合、プロンプト/レスポンスの全ペイロードをキャプチャ |

同じオプションは環境変数でも設定可能：`JJ_API_KEY`、`JJ_PROJECT`、`JJ_API_URL`。

---

## 6. 送信される情報（と送信されない情報）

デフォルトでは、JamJet Cloudは**メタデータ**をキャプチャします：モデル名、トークン数、レイテンシー、コスト見積もり、ツール名、ステータス。プロンプトとレスポンスの*内容*は、`capture_io=True`を設定しない限り送信**されません**。

SDKは**フェイルオープン**設計です：`api.jamjet.dev`に到達できない場合でも、エージェントは動作し続けます。失敗したバッチは指数バックオフで再試行され、5回連続で失敗すると破棄されます（サーキットブレーカー）。

## 7. 今後の予定

現在開発中です。詳細なタイミングについては[ロードマップ](https://jamjet.dev/cloud)をご覧ください。

- **マルチエージェント可視化** — エージェントID、エージェント間トレース伝播、エージェント間通信のネットワークグラフビュー（フェーズ1 / 2026年Q3）。
- **Java SDK** — Spring AI / LangChain4j向けの同様のドロップイン統合（フェーズ1）。
- **一元化されたポリシー強制** — サーバーサイドでのポリシー決定、監査された委任チェーン。
- **ホステッドメモリ** — Cloudに統合されたEngram、エージェントごとにスコープ化され、エージェントフリート全体で共有。
- **リプレイ** — ダッシュボードから入力記録を使用して任意のトレースを再実行。
- **OTel GenAI取り込み** — SDKの移行なしで、既存のPhoenix / OpenLLMetry / Langfuseで計装されたアプリをJamJetに接続。

## トラブルシューティング

| 症状 | 原因 |
|---|---|
| 呼び出しが表示されない | 5秒待ってください（フラッシュ間隔）。それでも表示されない場合は、APIキーがプロジェクトと一致しているか確認してください。 |
| `RuntimeError: JamJet Cloud not configured` | 最初のLLM呼び出しの前に`jamjet.configure()`が呼び出されませんでした。 |
| `BudgetExceeded` | 呼び出しによって累積支出が`budget(max_cost_usd=...)`を超えてしまいます。上限を増やすか削除してください。 |
| Anthropicの呼び出しがキャプチャされない | `pip install jamjet[anthropic]` — 自動パッチはSDKがインポート可能な場合にのみ適用されます。 |
| OpenAIの呼び出しがキャプチャされない | `pip install jamjet[openai]`。SDKは`OpenAI().chat.completions.create`をクラスレベルでパッチします — モジュールレベルとインスタンス使用の両方がキャプチャされます。 |
| APIから401エラー | キーがローテーションされたか間違っています。[app.jamjet.dev/dashboard/settings](https://app.jamjet.dev/dashboard/settings)で新しいプロジェクトを作成してください。 |
| APIから429エラー | レート制限を受けています。デフォルトはAPIキーあたり100リクエスト/秒です。制限の引き上げについてはDiscordまたは`support@jamjet.dev`までお問い合わせください。 |
