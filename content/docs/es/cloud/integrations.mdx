# Integraciones Cloud

Los SDKs de TypeScript y Python de JamJet Cloud funcionan con cualquier framework que use protocolos OpenAI / Anthropic. Para frameworks con sus propias abstracciones, las integraciones dedicadas te dan un control más limpio y telemetría más rica.
