# JamJet Cloud

JamJet Cloud는 AI 에이전트를 위한 호스팅형 거버넌스 레이어입니다. SDK 두 줄만 추가하면 정책 시행, 예산 추적, 승인 플로우, 감사 추적 기능을 모든 AI 앱과 프레임워크에서 사용할 수 있습니다.
